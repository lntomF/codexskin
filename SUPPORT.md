# CodeSkin 支持指南

> English version: [SUPPORT.md](docs/en/SUPPORT.md)

## 提问前请先检查

请检查最新 Release Note、已有 GitHub Issue、`README.md` 的故障排除内容，以及你的
Codex / ChatGPT Desktop 版本是否已知兼容。条件允许时，请正常重启 CodeSkin 与目标
应用后再复现问题。

## 应该在哪里提问

### Bug 或兼容性问题

当安装失败、CodeSkin 无法检测或连接目标应用、主题无法生效、还原失败，或者新版本
目标应用改变行为时，请使用对应模板提交 Issue。

### 功能建议

请说明用户问题、当前替代方案、受影响的用户群体，以及这个需求为什么不仅是个人的
视觉偏好。

### 主题投稿

仅当素材满足[素材政策](ASSET_POLICY.md)时，才可以使用主题投稿模板。

### 安全问题

不要创建公开 Issue。请阅读[安全政策](SECURITY.md)并使用私密漏洞报告渠道。

## 高质量支持请求应包含

CodeSkin、Windows 和目标应用版本；简短的复现步骤；预期与实际行为；以及脱敏后的
截图或日志。不要提交 API Key、Token、私有仓库路径、客户信息、私有任务/聊天内容，
或未授权第三方图片。

## 维护范围

CodeSkin 是独立的非官方项目。维护者会尽力支持可复现的、已支持版本中的问题。
