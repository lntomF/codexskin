# CodeSkin Code of Conduct

> 中文版：[查看中文行为准则](../../CODE_OF_CONDUCT.md)

## Our pledge

We want CodeSkin to be a welcoming, constructive project for users and
contributors of all experience levels.

## Expected behavior

- Be respectful and discuss ideas in good faith.
- Give specific, constructive feedback.
- Welcome newcomers and explain context when possible.
- Respect privacy, licenses, and the security boundaries of the project.
- Focus discussions on the project rather than personal attacks.

## Unacceptable behavior

- Harassment, discrimination, threats, or personal attacks.
- Publishing private data, credentials, logs, or personal information.
- Submitting unlicensed assets or knowingly infringing material.
- Spamming issues, discussions, or pull requests.
- Deliberately bypassing the project's security or contribution policies.

## Enforcement

Report serious conduct concerns through the repository's private
vulnerability-reporting channel. The maintainer may remove content, close
interactions, or restrict participation when necessary to protect the project
and its community.
