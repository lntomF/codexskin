# CodeSkin 社区行为准则

> English version: [CODE_OF_CONDUCT.md](docs/en/CODE_OF_CONDUCT.md)

## 我们的承诺

我们希望 CodeSkin 成为一个欢迎、建设性的项目，任何经验水平的用户与贡献者都可以
安心参与。

## 期待的行为

- 保持尊重，并以善意讨论想法。
- 提供具体、建设性的反馈。
- 欢迎新成员，并在可能时补充背景。
- 尊重隐私、许可证与项目安全边界。
- 聚焦项目问题，而不是进行人身攻击。

## 不可接受的行为

- 骚扰、歧视、威胁或人身攻击。
- 发布私密数据、凭据、日志或个人信息。
- 提交未授权素材或明知侵权的内容。
- 向 Issue、Discussion 或 PR 发送垃圾信息。
- 故意绕过项目的安全或贡献政策。

## 执行

严重的行为问题可通过仓库的私密漏洞报告渠道反馈。为保护项目和社区，维护者可以在必要
时移除内容、关闭互动或限制参与。
